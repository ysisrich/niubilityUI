# niubility-ui

## 使用 npm 安装

```js
npm install niubility-ui -S
```

### 全局引入

```js
import niubilityUi from "niubility-ui";
import "niubility-ui/dist/niubility-ui.css";
Vue.use(niubilityUi);
```

### 按需引入

```js
import { Icon, Button } from "niubility-ui";
import "niubility-ui/dist/niubility-ui.css";
Vue.use(Icon);
Vue.use(Button);
```

### 示例

```js
<nb-button type="primary" circle>
  主要按钮
</nb-button>
```
